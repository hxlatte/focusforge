# 🌿 FocusForge ⭐

A gamified productivity web app that helps users stay consistent, focused, and disciplined through task completion, Pomodoro focus sessions, study logs, and a virtual garden growth system.

---

## Introduction 

FocusForge is designed to make productivity engaging instead of stressful.  
Rather than using a basic to-do list, the app motivates users through visual rewards and focus tools.

This project helped me practice:
- Building a productivity system
- Implementing focus techniques (Pomodoro)
- Tracking daily learning progress
- Designing motivational user experiences

## Features 

### Garden Growth System
- Completing tasks grows plants and flowers
- Daily consistency builds a thriving garden
- Missed tasks slow garden growth

### Pomodoro Focus Timer
- Structured focus sessions for deep work
- Break reminders to avoid burnout
- Improvest concentration and productivity

### Study Logs
- Write daily study reflections
- Track learning consistency
- Build accountability and self-evaluation

### Task Management
- Add, edit, and delete tasks
- Mark tasks as completed
- Stay organized and focused

### Gamified Productivity
- Visual rewards increase motivation
- Encourages long-term habit building

---

## How It Works 

1. Add your daily tasks  
2. Start a Pomodoro focus session  
3. Complete tasks and stay consistent  
4. Log what you studied for the day  
5. Watch your garden grow as progress improves 🌱
